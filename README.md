# Cynexsys IT

Production ready multi page Next.js website for Cynexsys IT.

## Pages
- Home
- About
- Services
- Research
- Products
- Product detail pages
- Projects
- Project detail pages
- Framework
- Future
- Contact
- Founder
- Insights
- Investors
- Downloads
- AI Architecture
- Digital Twin Lab
- Research OS
- Trust Layer
- Moat
- Global Expansion

## Features
- shared header and footer across every page
- day and night mode toggle
- mobile friendly menu
- animated sections
- investor style future roadmap
- inquiry funnel and newsletter waitlist
- favicon, apple icon, open graph image, sitemap, robots, manifest

## Development
```bash
npm install
npm run dev
```

## Environment variables
Copy `.env.example` to `.env.local` and set:

- SMTP_HOST
- SMTP_PORT
- SMTP_USER
- SMTP_PASS
- CONTACT_EMAIL
